# 🐾 Приютик 

Сайт приюта для животных — каталог питомцев, заявки на усыновление и передачу, личный кабинет, панель администратора.

**Живой сайт:** [asyzhdayu.github.io/priyt](https://asyzhdayu.github.io/priyt/)

---

## Для посетителей

### Смотреть питомцев
Открой главную страницу — там все питомцы, которые ищут дом. Можно фильтровать по виду, полу, размеру и возрасту, а также искать по имени.

### Взять питомца домой
1. Нажми **«Взять домой»** на карточке питомца или перейди в раздел [Взять животное](https://asyzhdayu.github.io/priyt/adopt)
2. Выбери питомца из списка
3. Заполни контактные данные и расскажи о себе
4. Отправь заявку — ответим в течение 1–2 рабочих дней

> Подать заявку можно и без регистрации.

### Отдать животное в приют
Перейди в раздел [Отдать животное](https://asyzhdayu.github.io/priyt/surrender), заполни анкету о питомце и укажи причину. Животное появится в каталоге.

### Личный кабинет
После регистрации в разделе [Профиль](https://asyzhdayu.github.io/priyt/profile) доступны:
- Статус поданных заявок
- Список избранных питомцев
- Редактирование личных данных

## Для сотрудников приюта

### Вход в панель управления
Перейди на [/admin](https://asyzhdayu.github.io/priyt/admin) и войди под учётной записью сотрудника.

Демо-доступ: `admin@shelter.ru` / `admin123`

### Управление питомцами
- **Добавить питомца** — кнопка «+ Добавить питомца» в шапке панели
- **Редактировать** — иконка карандаша в строке питомца
- **Удалить** — иконка корзины (действие необратимо)
- **Фильтры** — поиск по имени и фильтр по статусу

### Заявки
Вкладка **Заявки** показывает все входящие запросы.
- ⏳ **На рассмотрении** — новые заявки
- ✓ **Одобрена** / ✕ **Отклонена** — обработанные

Нажми **«Детали»** чтобы увидеть полную информацию: карточку питомца с полом и породой, данные заявителя, тип жилья, наличие детей и других животных.

### Пользователи
Раздел **Пользователи** доступен только администратору. Здесь можно создавать аккаунты сотрудников, менять роли и удалять пользователей.
